# 插件安装

```sh
#github 
git clone --depth=1 https://github.com/USYDShawnTan/xiaotan-plugin.git ./plugins/xiaotan-plugin/
```

```sh
#gitee 
git clone --depth=1 https://gitee.com/xioatan/xiaotan-plugin.git ./plugins/xiaotan-plugin/
```

```sh
#安装依赖
npm install pnpm -g
pnpm install
```
